This is the library to use when using Robotic Sequencing RSi2cBoard to add functionnality
to your Arduino projects. 

Instructions to install. 

1-Download the archived library from github.

2-Extract it into your Arduino/library on your documents folder.

3-Rename the folder to RS. 

4-Start your Arduino dev. environment. Lock at sample. 

5-Look at tutorials on RoboticSequencing.com web site in the support or in the garden section for
users contributed code.

Enjoy,
